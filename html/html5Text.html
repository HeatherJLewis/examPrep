Text is easier to understand when it is split up into paragraphs.

Semantic MarkUp

Strong

Strong is used to show emphasis and importance. Shown by default as bold.

em is used to show emphasis that subtly changes the meaning of the sentence. Shown by default in italic.

A lovely little quote in a block quote. Browsers tend to indent this but you shouldn't use this just to indent text - use CSS instead. Can add a link as a citation in the tag as an attribute.

A quote tag adds quotation marks and can be used within a p tag. As a wise person said, stay happy!

An abbr tag enables you to specify the full term if you use an abbreviation or acronym. For example: Prof Jane Mellor. But what is the point of this? Does it help screen readers?

You can create a citation: A Brief History of Time by Stephen Hawking sold a lot of books!

A definition can also be created thus: A black hole is a very dense region of space.
